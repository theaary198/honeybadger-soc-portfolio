# HB-0002 — SSH Authentication Abuse Analysis

## Summary
This project focuses on repeated successful SSH authentication using weak/default credentials. The investigation began as a brute-force hypothesis but was correctly reclassified as valid account abuse due to the absence of failed login events.

## Skills Demonstrated
- Authentication log analysis
- Hypothesis correction
- Valid account abuse assessment
- Escalation reasoning
- Interview-ready communication

## Included Files
- `triage_notes.txt`
- `interview_answer.txt`

## Key Takeaway
This case demonstrates the importance of correcting assumptions and distinguishing brute-force activity from successful weak credential abuse.
