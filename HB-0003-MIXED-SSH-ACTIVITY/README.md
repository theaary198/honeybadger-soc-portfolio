# HB-0003 — Mixed SSH Activity Triage

## Summary
This project analyzes a mixed SSH dataset containing both benign and malicious sessions. Sessions were classified, prioritized, and assessed for severity using SOC-style triage logic.

## Skills Demonstrated
- Session classification
- Benign vs malicious differentiation
- Priority-based investigation
- Severity assessment
- SOC reporting

## Included Files
- `final_soc_report.txt`

## Key Takeaway
This case demonstrates that not every SSH session is malicious and highlights the importance of behavioral context and analyst judgment.
