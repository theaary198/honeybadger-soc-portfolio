# HB-0001 — Local SSH Recon Investigation

## Summary
This project documents a Cowrie honeypot SSH intrusion simulation involving weak credential login, reconnaissance activity, file download behavior, execution attempts, IOC extraction, MITRE ATT&CK mapping, and analyst reporting.

## Skills Demonstrated
- Log triage
- IOC extraction
- Attack chain reconstruction
- MITRE ATT&CK mapping
- Analyst reporting

## Included Files
- `final_case_report.txt`
- `advanced_case_report.txt`

## Key Takeaway
This case demonstrates how to investigate a basic SSH intrusion and convert raw logs into a structured SOC-style case report.
